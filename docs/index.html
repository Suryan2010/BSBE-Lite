Installation

  • For MCBE 1.18.12 just download the resourcepack and install it like other addons/resourcepack
  • For MCBE 1.18.30 - latest stable (Renderdragon Engine)
    1. Download the resourcepack and materials file
    2. Install some material files, here is how to install material files on Windows and on Android
    3. Install the resourcepack
    4. Enjoy :)

  • For MCBE with Renderdragon engine, don't forget to install the resourcepack too!. Its needed for activate sky, cloud, and fix water detection!
